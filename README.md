## Usage

1-- Make config/configdata.php file with content:

```php
<?php

return [
    'database_type' => 'mysql',
    'database_name' => 'dbnamehere',
    'server' => 'localhost',
    'username' => 'usernamehere',
    'password' => 'passwordhere'
];
```
2-- $ composer require catfan/medoo

3-- $ composer require nikic/fast-route
